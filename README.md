# CL-STL
Common Lisp library like C++ STL ( Standard Template Library ).

CL-STL depend on CL-OVERLOAD and CL-OPERATOR.

CL-STL provids...

* Containers ( vector, deque, list, forward-list, etc. )
* Algorithm
* Functors
* Iterators
* etc.

Not provides...

* some containers ( unordered-map etc. )
* stream features
* etc.
