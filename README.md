# CL-STL
Common Lisp library like C++ STL ( Standard Template Library ).

#### This library depends on...

* closer-mop
* CL-OVERLOAD
* CL-OPERATOR


#### CL-STL provides...

* Containers ( vector, deque, list, forward-list, map, set, etc. )
* Algorithms
* Functors
* Iterators
* etc.

#### Not provides...

* Hash containers ( now writing... )
* Stream features
* etc.
